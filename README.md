# dynamic_pomodoro
This program will dynamically calculate breaks for you based on how much you have worked and how much break time you want per work time. For example, let's say you want to have a break of 1/5th the amount you work. After 30 minutes of work then you could take a 6 minute break, or after 1 hour of work you could take a 12 minute break.

The layout consists of a "stopwatch", a "break fraction" selector, and a break countdown timer, along with some buttons for control. The stopwatch allows you to time how long you have worked, and the break fraction selector allows you to select how much break time you want per work time. Once the user stops the stopwatch, the app will display the amount of break time you have accumulated along with one button to continue working and one to start your break. If the user starts their break, then a countdown timer starts showing how much break time they have left. If the user selects to continue working, then the stopwatch starts again from where it last left off.
